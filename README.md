# youtube-analysis
YouTube trending data analysis using Python and NLP to study engagement patterns and audience sentiment
# YouTube Data Analysis

## Objective
To analyze trending YouTube data and understand factors influencing video popularity and user engagement.

## Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, NLTK

## Key Analysis
- Relationship between views, likes, and comments  
- Sentiment analysis on user comments  
- Category-wise analysis of trending videos  
- Time-based trends in video performance  

## Key Insights
- Higher views are generally associated with higher likes and comments  
- Audience sentiment is mostly positive across trending videos  
- Certain categories consistently perform better  
- Engagement metrics are strongly correlated  

## Dataset
YouTube trending dataset

## Conclusion
Engagement metrics and content category play a major role in determining video popularity and audience response.
